# 연수담 — 초등교사 연수관리

대한민국 초등교사가 해마다 이수하는 법정·의무연수와 개인 연수를 한곳에서 점검하고 기록하는 웹앱입니다.

## 주요 기능

- 연도별 의무연수 기본 목록 자동 생성
- 연수 추가·수정·삭제와 삭제 되돌리기
- 시작 전·진행 중·이수 완료·해당 없음 상태 관리
- 기준 시간, 이수 시간, 기관, 날짜, 방식, 메모 기록
- 제목·기관·메모 검색과 구분·상태·분류 필터
- 전년도 개인 연수 가져오기
- 전체 연도 JSON 백업·복원
- 현재 연도 CSV 저장과 A4 인쇄·PDF 저장
- PC·태블릿·모바일 반응형 화면

> 기본 목록은 점검을 돕는 참고용입니다. 실제 이수 대상, 시간, 인정 방법은 해당 연도 학교와 소속 교육청의 안내를 우선 확인하세요.

## Mac에서 실행하기

터미널을 열고 프로젝트 폴더로 이동한 뒤 아래 명령어를 한 줄씩 실행합니다.

```bash
npm install
```

필요한 실행 도구를 설치합니다. 처음 한 번 또는 `package.json`이 바뀌었을 때 실행합니다.

```bash
npm run dev
```

개발용 웹사이트를 실행합니다. 터미널에 표시된 `http://localhost:3000` 같은 주소를 브라우저에서 엽니다.

실행을 멈추려면 터미널에서 `Control + C`를 누릅니다.

## 테스트하기

```bash
npm test
```

완성본을 만들 수 있는지 확인하고, 첫 화면·한글 설정·공유 이미지를 자동 점검합니다.

직접 확인할 항목:

1. 현재 연도 기본 연수 목록이 보이는지 확인합니다.
2. `연수 추가`를 눌러 개인 연수를 저장합니다.
3. 새로고침한 뒤 방금 적은 기록이 그대로인지 확인합니다.
4. 완료 동그라미를 눌러 진행률과 이수 시간이 바뀌는지 확인합니다.
5. 다른 연도로 이동했을 때 기록이 섞이지 않는지 확인합니다.
6. `데이터 관리 → 전체 연도 백업`으로 JSON 파일이 저장되는지 확인합니다.
7. 모바일 폭에서 글자가 잘리지 않고 자연스럽게 줄바꿈되는지 확인합니다.
8. `데이터 관리 → 인쇄·PDF 저장`에서 A4 목록이 보기 좋게 나오는지 확인합니다.

## 개인정보와 저장 위치

입력한 자료는 별도 서버로 전송하지 않고 현재 브라우저의 로컬 저장소(`localStorage`)에만 저장합니다. 로컬 저장소는 쉽게 말해 “이 브라우저 안에 있는 작은 기록 보관함”입니다.

- 브라우저 기록을 삭제하거나 기기를 바꾸면 자료가 사라질 수 있으므로 정기적으로 JSON 백업을 내려받으세요.
- 같은 컴퓨터와 브라우저를 쓰는 사람은 기록을 볼 수 있으므로 공용 PC 사용을 권장하지 않습니다.
- 학생 실명, 상담 내용, 연락처 등 개인정보를 메모에 입력하지 마세요.
- 수료증 파일 자체는 저장 공간과 개인정보 보호 문제 때문에 웹앱 안에 보관하지 않습니다.

## 자주 생기는 오류

```text
오류 메시지: npm: command not found
발생 원인: Node.js가 설치되지 않았습니다.
해결 방법: Node.js 22.13 이상을 설치한 뒤 터미널을 다시 엽니다.
확인할 파일: package.json의 engines 항목
```

```text
오류 메시지: Port 3000 is in use
발생 원인: 다른 앱이 3000번 주소를 사용 중입니다.
해결 방법: 오류가 아니라면 터미널에 표시된 3001, 3002 같은 새 주소로 접속합니다.
확인할 파일: 수정할 파일 없음
```

```text
오류 메시지: 저장한 기록이 보이지 않음
발생 원인: 다른 브라우저를 열었거나 브라우저 데이터를 삭제했을 수 있습니다.
해결 방법: 같은 브라우저인지 확인하고, 백업 JSON 파일이 있으면 데이터 관리에서 불러옵니다.
확인할 파일: components/TrainingManager.tsx
```
